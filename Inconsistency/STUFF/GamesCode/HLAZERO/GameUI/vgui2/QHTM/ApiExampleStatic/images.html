Image Support

QHTM Supports the following images formats:

  • BMP
  • MNG
  • PCX
  • JPEG
  • GIF

QHTM uses a custom library called ImgLib, which means you get the full source code to ImgLib too. ImgLib is a powerful and flexible too for loading and displaying images and it can be used in your applications directly.

QHTM Light supports only images of type BMP, so if you are using the light version you will see what QHTM does when it can't load an image.
